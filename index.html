Pertanyaan Umum Seputar Deploy

❓ Apakah benar-benar GRATIS? Tidak ada biaya tersembunyi?
Ya, 100% GRATIS untuk basic usage!

• Vercel Free: 100GB bandwidth, 100GB storage, unlimited sites
• Netlify Free: 100GB bandwidth, 300 build/month
• GitHub Pages: Unlimited semuanya!

Cukup untuk website dengan traffic ribuan pengunjung/hari. Baru bayar kalau sudah skala besar (enterprise).
❓ Berapa lama sampai situsnya online?
Sangat cepat!

• Upload ke GitHub: 1-5 menit (tergantung ukuran file)
• Deploy di Vercel/Netlify: 30-90 detik otomatis
• Domain aktif: INSTAN setelah deploy selesai

Total waktu dari nol sampai live: kurang dari 10 menit!
❓ Kalau saya update file, harus deploy ulang?
TIDAK PERLU manual lagi! Itu kelebihan modern hosting:

1. Edit file di komputer Anda
2. Push ke GitHub (git push)
3. Vercel/Netlify OTOMATIS detect perubahan
4. Re-deploy sendiri dalam 1 menit
5. Update live tanpa Anda sentuh sama sekali!

Proses ini disebut "Continuous Deployment" (CD).
❓ Apakah aman? Data user tidak bocor?
Sangat aman untuk frontend/static site!

✅ HTTPS/SSL gratis (enkripsi data)
✅ DDoS protection built-in
✅ Global CDN (cepat & reliable)
✅ Regular security updates dari provider

Catatan: Untuk backend API (WhatsApp integration), perlu setup tambahan seperti environment variables & database security.
❓ Bisa pakai domain sendiri (misal: ai.situsku.com)?
BISA! Dan GRATIS juga di beberapa platform!

• Vercel: Gratis custom domain, tinggal pointing DNS
• Netlify: Gratis custom domain
• GitHub Pages: Gratis custom domain

Caranya: beli domain (±Rp 120rb/tahun di Niagahoster/Namecheap), lalu tambahkan di settings platform hosting. Selesai!
